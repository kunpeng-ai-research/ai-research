# 图片资源目录

本目录存放OpenClaw PKM案例的所有配图。

## 文件列表

- `openclaw-pkm-cover.png` - 博客封面图 (1200×630)
- `architecture.png` - 系统架构图 (Mermaid格式)
- `screenshots/` - 实际运行截图
  - `feishu-bot.png` - 飞书机器人交互
  - `daily-report.png` - 每日报告示例
  - `memory-search.png` - 语义搜索演示

## 使用说明

### 生成封面图

运行脚本生成标准尺寸封面：

```bash
cd scripts
python3 generate-cover.py
```

### 截图要求

- 格式: PNG
- 分辨率: 至少 1920×1080
- 命名: 小写+连字符，如 `feishu-bot.png`
